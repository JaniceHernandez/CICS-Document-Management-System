# **App Name**: CICS Docs Hub

## Core Features:

- Secure Student Authentication: Authentication via NEU Google accounts with domain validation (@neu.edu.ph).
- Admin Document Management: Admins can upload, categorize, edit, and delete PDF documents. Drag-and-drop upload supported.
- Student Document Access: Students can search, filter (by keyword, program, category), preview, and download documents.
- Admin Student Management: Admins can view student lists, add/block accounts, and monitor undergraduate programs.
- Analytics Dashboard: Admins can track login and download statistics, visualize trends with charts, and export reports (CSV/PDF).
- Inquiry Period Management: Admins can configure inquiry period dates, enable/disable submissions, and post inquiry-related announcements.
- AI-Powered Document Summarization: Students can use a tool that generates a brief summary of any PDF document using AI to understand the key points quickly.

## Style Guidelines:

- Primary color: Navy blue (#00308F) to convey trust, security, and institutional authority. The color is seen as formal, solid and professional.
- Background color: Light gray (#F0F4F8) for a clean, modern, and uncluttered look.
- Accent color: Gold (#FFD700) for highlights and calls to action, subtly referencing academic achievement.
- Headline font: 'Poppins', a geometric sans-serif for a precise and contemporary look.
- Body font: 'PT Sans', a humanist sans-serif providing a modern yet accessible reading experience.
- Use a consistent set of clean, professional icons throughout the app, in a filled style.
- Use a grid-based layout for a structured and responsive design.
