# 📄 CICS Document Management App

## 📌 Overview
The **CICS Document Management App** is a web-based platform developed for the **College of Informatics and Computing Studies (CICS) at New Era University (NEU)**.  
It provides a **centralized system for storing, organizing, and accessing official PDF documents**.

The system supports **two user roles: Admin and Student** and uses **Google OAuth authentication restricted to `@neu.edu.ph` email accounts** to ensure that only authorized NEU students can access the platform.

---
### 💻 Application Link
https://document-management-system-gules.vercel.app/ 

---

## 👥 User Roles

### 🧑‍🎓 Student
Students can:
- Log in using their **NEU Google account**
- Set their **undergraduate program** on first login
- **Search and filter documents**
- **Preview and download PDF files**
- View **announcements and inquiry period notices**
- Receive notifications if their account is **blocked**

---

### 🛠️ Admin
Admins can:
- **Upload, edit, and delete PDF documents**
- Categorize documents by **program, department, batch, or type**
- **View and manage student accounts**
- **Block or unblock users**
- Monitor **login and download statistics**
- Generate **usage reports**
- Manage **inquiry periods and announcements**

---

## ⭐ Key Features
- Google OAuth authentication (`@neu.edu.ph` only)
- Document search and filtering
- PDF preview and download
- Student account management
- Usage analytics dashboard
- Inquiry period configuration
